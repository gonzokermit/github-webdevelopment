Als Administrator in Windows anmelden

Befehlszeile lokal

runas /user:Rechnername\administrator Programmdatei ---> Kennwort eingeben

z.B. runas /user:pc-123\administrator cmd.exe ---> Kennwort eingeben

Befehlszeile Domain

runas /user:Domain\administrator \\Server\Verzeichnis\Programmdatei ---> Kennwort eingeben

z.B. runas /user:ads\administrator \\192.168.55.123\itservce\cmd.exe ---> Kennwort eingeben

Windows Netzwerkbefehle

Konfiguration TCP/IP-Netzwerk

ipconfig /all

Routing anzeigen

route print

Eine satische Route hinzufuegen

route add Ziel-Netzwerk Subnetzmaske Gateway-Rechner-IP

z.B. route add 192.168.55.0 255.255.255.0 192.168.55.1

Soll die Route persistent eingetragen werden

route add -p Ziel-Netzwerk Subnetzmaske Gateway-Rechner-IP

z.B. route add -p 192.168.55.0 255.255.255.0 192.168.55.1

Eine statische Route entfernen

route delete Ziel-Netzwerk

z.B. route delete 192.168.55.0

Belegte Ports anzeigen

netstat -ano | find /i "Abhoeren" oder "Listen"
